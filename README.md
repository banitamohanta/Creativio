🎨 Creativio – Design Landing Page

Creativio is a modern and responsive landing page that transforms rough sketches into polished, editable designs. It focuses on clean UI/UX, smooth layout, and an engaging user experience.

🚀 Features
✨ Clean and modern UI design
📱 Fully responsive layout
🎯 Sketch-to-design concept presentation
🌐 Multiple sections (Home, Features, How It Works, Contact)
⚡ Smooth and user-friendly interface

🛠️ Tech Stack
HTML5
CSS3
JavaScript
